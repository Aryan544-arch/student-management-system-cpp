Student Management System (C++)

Overview

A menu-driven Student Management System developed in C++. The application allows users to add, view, search, update, and delete student records using a vector-based storage system. The project demonstrates Object-Oriented Programming (OOP), encapsulation, constructors, and CRUD operations in a multi-file C++ application.

Features

- Add Student Records
- View All Student Records
- Search Students by Name
- Update Student Marks
- Delete Student Records
- Input Validation for Student Names
- Menu-Driven Interface

Concepts Used

- Classes and Objects
- Encapsulation
- Constructors
- Vectors (STL)
- Functions
- Loops
- Switch Statements
- Header and Source File Separation
- Basic Input Validation
